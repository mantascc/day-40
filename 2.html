Audio Sketches

A series of explorations into mapping sound to visual properties.

0 — Direct

One value → one property.

Amplitude directly maps to circle radius. No filtering, no transformation. The raw signal drives the visual state 1:1.

1 — Scaled

One value → one property, with range mapping.

Amplitude (0–255) is remapped to Opacity (0.1–1.0). The visual output is constrained to a useful aesthetic range.

2 — Threshold

Value triggers state change.

If amplitude > 180, the state is "Flash". Below, it is "Dark". A binary reaction to an analog signal.

3 — Smoothed

Value filtered over time.

We linear interpolate (Lerp) toward the current amplitude. The circle "breathes" rather than twitches. It has momentum.

4 — Derived

Computed property from raw data.

We calculate the rate of change (current - previous). Visuals react to the derivative—the "impulse"—rather than the volume.

5 — Cross-mapped

Multiple inputs → multiple outputs.

Amplitude → size. Rate of change → rotation speed. A system of relationships creates a more complex behavior.

6 — Emergent

Sound feeds rules, not properties.

Amplitude spawns particles. Silence applies gravity. The sound shapes a system that runs on its own internal logic.

End of Sketches.
